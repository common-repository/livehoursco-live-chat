Add Live Chat Support for your website (messages come to your phone a text messages).

== Description ==

OnlineLiveChat.com is a live chat support tool for small business owners. When you add a OnlineLiveChat live support chat to your website, your customers will be amazed at your instant, educated, world-class response... they won't have a clue that you're responding to them via text message. For more details, check out http://www.onlinelivechat.com.

This plugin makes it easy to install OnlineLiveChat on your Wordpress site. 
1. Install the plugin.
1. Enter your OnlineLiveChat token.

== Installation ==

1. Search for `OnlineLiveChat` in the 'Plugin Gallery' and use the one-click install *OR* [Download](https://github.com/dallas22ca/OnlineLiveChatWP) the OnlineLiveChat.com plugin and Upload the `livehours` folder to the `/wp-content/plugins/` directory in your own WordPress installation.
